Java Reflection

发表于 2020-08-05 | 更新于: 2020-08-11

Java反射从入门到放弃。

前言

Java的反射是指程序在运行期可以访问、检测和修改它本身状态或行为的一种能力。

获取Class对象

常用有三种方式获取Class对象

Class.forName

classname.class

Field API

一个Field对象包含了一个成员变量的所有信息：

  • getName():返回成员变量名称。
  • getType():返回成员变量类型。
  • getModifiers():返回成员变量修饰符。

  • set(Object obj, Object value):设置成员变量值。
  • get(Object obj):返回成员变量值。

Class类提供的newInstance()方法，是通过调用该类的 public 无参数构造方法 创建的实例，

如果构造方法带有参数，或者构造方法的 作用域 不是public，就无法直接通过Class.newInstance()来创建新的实例对象。

为了调用任意的构造方法，Java的反射API提供了Constructor对象，它包含一个构造方法的所有信息，也可以用来创建一个新的实例。

获取Constructor实例

通过Class实例获取Constructor的方法如下：

  • getConstructor(Class...)：返回类的某个public的Constructor；
  • getDeclaredConstructor(Class...)：返回类的某个Constructor；
  • getConstructors()：返回类的所有public的Constructor；
  • getDeclaredConstructors()：返回类的所有Constructor。

获取Method对象

通过反射，获取类中 包含的 成员方法信息。

Class类提供了以下几个方法来获取Method对象：

Method getMethod(name, Class...)：返回指定名称的Method对象，作用域由public修饰（包括父类）
Method getDeclaredMethod(name, Class...)：返回当前类中指定名称的Method（不包括父类）
Method[] getMethods()：返回所有public修饰的Method（包括父类）
Method[] getDeclaredMethods()：返回当前类的所有Method（不包括父类）

与获取Field对象时 提供的方法 类似。

Method API

一个Method对象包含一个方法的所有信息：

getName()：返回方法名称，
getReturnType()：返回方法返回值类型，也是一个Class实例，例如：String.class；
getParameterTypes()：返回方法的参数类型，是一个Class数组，例如：[String.class, int.class]；
getModifiers()：返回方法的修饰符，它是一个int，不同的bit表示不同的含义。
